# permit2drive-pages
Documentation for Permit 2 Drive app

# File Types
.html (mostly web style):
    about.html
    faq.html
    index.html
    state-requirements.html
.md (mostly document style):
    legal.md
    privacy.md

# Running Viewer
- in terminal run: bundle exec jekyll serve --livereload
- Open browser to http://localhost:4000 (or differnt server address as on terminal)
